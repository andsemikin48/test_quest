---Документация к программе генерации отчетов по сотрудникам---
Общее описание
Программа для анализа CSV-файлов с данными сотрудников и генерации различных отчетов.
Читает один или несколько файлов и выводит структурированные отчеты в виде таблиц.

Установка зависимостей.
При создании использована библиотека tabulate:
Для установки использовать "pip install tabulate"

Формат запуска:
    python report_csv.py --files employees1.csv employees2.csv --report performance

    Базовый синтаксис:
    python report_csv.py --files ФАЙЛ1 [ФАЙЛ2 ...] --report ТИП_ОТЧЕТА

Файлы формата CSV для формирования отчета помещать в папку files в папке со скриптом.

Согласно ТЗ создана функция формирования отчета:
    performance - Отчет по эффективности
    Описание: Средняя оценка эффективности по должностям
    Сортировка: По убыванию эффективности

Формат входных данных:
Файл CSV должен содержать строку с заголовками:
    name,position,completed_tasks,performance,skills,team,experience_years
    Обязательные колонки:
    position - Должность
    performance - Оценка эффективности (число)
